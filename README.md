# Movbase

Movbase is a simple and responsive web app for browsing **trending movies and TV shows**, built using **Vanilla JavaScript and Vite**, and powered by the **TMDB API**.

It allows users to explore trending content, view detailed information, and watch trailers directly inside the app.


## Features

- Trending Movies & TV Shows
- Movie / TV Show Detail Page
- Trailer Playback (YouTube Embed)
- Responsive Navigation Bar
- Mobile Friendly UI
- Modular JavaScript Architecture
- Fast build & dev environment using Vite


## Tech Stack

- HTML5  
- CSS3  
- Vanilla JavaScript (ES Modules)  
- Vite  
- TMDB API  
- Font Awesome Icons


## Environment Variables

Create a `.env` file in the root directory and add:

```env
VITE_API_URL=https://api.themoviedb.org/3
VITE_API_TOKEN=your_tmdb_bearer_token
VITE_POSTER_URL=https://image.tmdb.org/t/p
```
